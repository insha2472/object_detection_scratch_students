# Student Portfolio (Starter)

This is a simple, single-page portfolio template intended for students.

How to customize:
- Open `index.html` and replace **Your Name**, bio, and contact email.
- Add or edit project entries in `main.js` under the `projects` array.
- Update styles in `styles.css`.

Preview locally:
- Use VS Code Live Server extension or open `index.html` in your browser.

Deploy:
- Host via GitHub Pages: push the `stu_144` folder to a repo and enable Pages, or drop files into a static hosting provider.

Tips:
- Replace placeholder links with actual project URLs or GitHub repos.
- Add images under a new `images/` folder and reference them from `index.html` or `main.js`.
